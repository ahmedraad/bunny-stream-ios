# Bunny Stream iOS – Example App

This is the iOS demo application showing how to integrate and use the Bunny Stream package.

## Prerequisites

- Xcode 13.0 or later
- Swift 5.9

## Getting Started

1. **Clone the repository**

   ```bash
   git clone https://github.com/BunnyWay/bunny-stream-ios.git
   ```

2. **Navigate to the Example App**

   ```bash
   cd bunny-stream-ios/Example-App
   ```

3. **Open the Xcode project**

   Double-click the `Example-App.xcodeproj` file or open it via terminal:

   ```bash
   open Example-App.xcodeproj
   ```

4. **Build & Run**

   - Select an iOS simulator or a physical device.
   - Press `⌘R` to build and run the Example App.

## What’s Inside

- **Example App**: A sample SwiftUI application demonstrating the use of the Bunny Stream packages:
  - Initializing `BunnyStreamAPI`
  - Playing videos using `BunnyStreamPlayer`
  - Uploading videos with `BunnyStreamUploader`
  - Recording and uploading with `BunnyStreamCameraUpload`

Feel free to explore the code and use it as a reference for your own implementation.
